## Cisco Webex module

This module connects to Cisco Webex devices

### Call actions

- Dail
- Disconnect
- Accept
- Auto Answer Mode
- Auto Answer Mute
- Auto Answer Delay

### Audio actions

- Set volume of device
- Mute/Unmute Microphones
- Microphone input
- Music mode on/off
- Noise removal on/off

### Presentation

- Start/Stop

### Camera

- Set Main Video Source
- Activate preset
- Camera Position Set
- Video Matrix Reset
- Trigger Auto Focus
- Camera Ramp
- Camera Background

### Video

- Monitors
- Output Monitor Role

### Conference

- Do Not Disturb
- Selfview

### Custom actions

- Custom xConfiguration
- Custom xCommand
- DTMF Send
- OSD Key Click
- OSD Key Press
- OSD Key Release
- Send Message

### Other

- Stand-by
